## 📌 Project Overview  
This repository contains my Digital Literacy Project completed as part of the CSE0001 course at VIT Bhopal University. In this project, I have worked as a *Student Digital Ambassador* to explore and demonstrate essential digital skills required for academic and professional growth.

The project covers five key areas: digital awareness, professional portfolio building, coding and collaboration tools, communication etiquette, and cybercrime awareness. Each task is designed to develop practical knowledge and responsible digital behavior.
